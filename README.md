# Shorts Downloader

[![Live Demo](https://img.shields.io/badge/demo-live-brightgreen)](https://shorts-dwnldr.onrender.com)

Téléchargez et republiez des vidéos courtes (YouTube Shorts, Instagram Reels, TikTok) directement depuis votre navigateur.

## Utilisation

1. Collez l'URL d'une vidéo
2. Prévisualisez les infos
3. Téléchargez ou republiez sur YouTube

## Lancer en local

```bash
git clone https://github.com/tcacciatore/shorts_dwnldr.git
cd shorts_dwnldr
pip install -r requirements.txt
python app.py
```

Accédez ensuite à [http://localhost:5001](http://localhost:5001).

> Pour la fonctionnalité de republication YouTube, ajoutez votre fichier `client_secrets.json` à la racine du projet.
